### 🧩 Challenge 1 – **"Kalkulator Mini"**

**📖 Cerita**:
Kamu diminta membuat kalkulator sederhana yang bisa membantu anak SD menghitung hasil dua bilangan dengan operator dasar.

**🎯 Tugas**:

* Input dua angka dan satu operator (`+`, `-`, `*`, `/`)
* Cetak hasil operasi

**⚙️ Requirement**:

* Gunakan `Double` dan `when` expression
* Validasi operator

**✅ Ekspektasi Output**:

```
Masukkan angka pertama: 10  
Operator: *  
Masukkan angka kedua: 3  
Hasil: 30.0
```

### 🧩 Challenge 2 – **"Ganjil atau Genap?"**

**📖 Cerita**:
Kamu membuat sistem pemilihan kursi bioskop yang ingin menampilkan apakah kursi bernomor ganjil atau genap.

**🎯 Tugas**:

* Input satu angka (Int)
* Tampilkan apakah angka itu ganjil atau genap

**⚙️ Requirement**:

* Gunakan operator modulus `%`

**✅ Ekspektasi Output**:

```
Nomor kursi: 17  
Status: Ganjil
```

### 🧩 Challenge 3 – **"Diskon untuk Semua!"**

**📖 Cerita**:
Toko online-mu memberi diskon jika pembelian di atas 100.000. Buat sistem hitung harga akhir berdasarkan diskon 10%.

**🎯 Tugas**:

* Input harga belanja (Double)
* Hitung diskon 10% jika harga > 100.000
* Tampilkan harga akhir

**⚙️ Requirement**:

* Gunakan `if` dan `*`, `-`

**✅ Ekspektasi Output**:

```
Total belanja: 120000  
Diskon: 12000  
Total bayar: 108000
```

### 🧩 Challenge 4 – **"Siapa yang Lebih Besar?"**

**📖 Cerita**:
Kamu membuat alat sederhana untuk membandingkan dua angka. Gunakan untuk seleksi nilai ujian.

**🎯 Tugas**:

* Input dua angka
* Tampilkan angka yang lebih besar

**⚙️ Requirement**:

* Gunakan `if-else`

**✅ Ekspektasi Output**:

```
Angka A: 70  
Angka B: 85  
Angka lebih besar: 85
```

### 🧩 Challenge 5 – **"Masuk atau Tidak?"**

**📖 Cerita**:
Seseorang hanya boleh masuk club malam jika usianya lebih dari 21 tahun dan punya kartu member.

**🎯 Tugas**:

* Input umur (Int) dan status member (Boolean)
* Cek dan tampilkan apakah user boleh masuk

**⚙️ Requirement**:

* Gunakan `&&` dan `Boolean`

**✅ Ekspektasi Output**:

```
Umur: 23  
Member: true  
Status: Boleh masuk
```
